수동 WebRTC 연결 방법

- A “Create Offer” 클릭 → A SDP Offer 텍스트 생성
- A SDP Offer 복사 → B SDP Offer 붙여넣기 → B “Create Answer” 클릭 → B SDP Answer 텍스트 생성
- B SDP Answer 복사 → A SDP Offer 붙여넣기 → A “Set Remote Answer” 클릭 → A ICE Candidates 텍스트 생성
- A ICE Candidates 복사 → B Remote ICE Candidates 붙여넣기 → B “Add Remote ICE Candidates” 클릭
